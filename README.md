# DropShot
My water droplet projekt on Raspberry PI

Programm access DropShoot.txt and read commands (action, signal, time).

Action:
  - c = Camera
  - w = wait
  - d1 = Drop 1 / Valve 1
  - d2 = Drop 2 / Valve 2
  - d12 = Drop 1 and 2 / Valve 1 and 2
  - f = Flash
Signal:
  - 1 = signal on
  - 0 = signal off
Time:
  in milliseconds
